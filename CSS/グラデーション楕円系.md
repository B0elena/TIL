## グラデーションを実現する2つの代表的なbackgroundの値
|値|説明|
|-|-|
|linear-gradient()|直線的グラデーションを作成できる。|
|radial-gradient()|楕円系のグラデーションを作成できる。|

## radial-gradient()の使い方
### 基本
 ```css
 background: radial-gradient(中央の色, 外側の色);
 ```
 ```css
 background: radial-gradient(#05FBFF, #1E00FF);
 ```
