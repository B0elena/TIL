## 変数と定数
``` swift
// 変数
var a = 1
// 定数
let b = 2
```

## データ型
Int型：整数<br>
UInt型：正の整数<br>
Float型：不動小数点<br>
Double型：不動小数点<br>
String型：文字列<br>
Bool型：真理値<br>
``` swift
var a:Int = 1
var b:Double = 3.14
var c:String = "Hello"
// type(of)で型を確認できる
print(type(of: a))
```

## 演算
演算子（オペレータ）　：変換の決まり（+,-,＊,/,%)
被演算子（オペランド）　：変換元データ
``` swift
var a:Int = 4
var b:Int = 2
var c:Int
c = a + b
c = a - b
c = a * b
c = a / b
c = a % b
```
